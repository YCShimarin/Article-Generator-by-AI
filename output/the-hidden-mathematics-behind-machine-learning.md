## Linear Algebra Fundamentals

Linear algebra is a cornerstone of machine learning. It provides the mathematical tools to represent and manipulate data effectively. At its core, it deals with vectors and matrices – fundamental building blocks for algorithms.

A **vector** is simply a list of numbers. Think of it as a point in space. For example, the vector [2, 3] represents a point two units along the x-axis and three units along the y-axis.  We often use vectors to represent features of a data point, like height and weight.

**Matrices** are rectangular arrays of numbers. They’re used to represent relationships between vectors.  A matrix can be thought of as a transformation that acts on vectors.  For instance, a matrix can scale or rotate data.

$$
A = \begin{bmatrix}
1 & 2 \\
3 & 4
\end{bmatrix}
$$

Here, *A* is a 2x2 matrix.  Multiplying a matrix by a vector changes the vector’s representation.  These operations are crucial for tasks like dimensionality reduction and feature extraction, which are vital in many machine learning models.

## Probability and Statistics Concepts

Machine learning relies heavily on probability and statistics. These concepts provide the foundation for understanding how algorithms learn from data. Let’s explore some key ideas.

At its core, machine learning aims to find patterns in data. Probability helps us quantify the likelihood of different outcomes. For example, a spam filter uses probability to determine if an email is likely to be spam based on words and phrases. 

*   **Probability Distributions:** These describe the likelihood of different values occurring. A common one is the normal distribution, often represented as a bell curve.
*   **Bayes’ Theorem:** This is crucial for updating beliefs as new data arrives. It’s expressed as:

    $$
    P(A|B) = \frac{P(B|A)P(A)}{P(B)}
    $$

    Where:
    *   $P(A|B)$ is the probability of event A happening given that event B has happened.
    *   $P(B|A)$ is the probability of event B happening given that event A has happened.
    *   $P(A)$ is the prior probability of event A.
    *   $P(B)$ is the probability of event B.

Understanding these concepts is essential for grasping how machine learning algorithms make predictions and decisions.
